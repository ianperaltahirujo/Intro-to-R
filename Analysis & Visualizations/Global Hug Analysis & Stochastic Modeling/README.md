# Intro-to-R: Global Hub Analysis & Stochastic Modeling

## About This Project
This repository contains a reproducible data science report built with Quarto.
The project analyzes annual passenger traffic across six major global airports
(2020–2025) and demonstrates Monte Carlo numerical integration using a Weibull
probability density function. It also explores how structured prompting of
generative AI tools produces higher-quality outputs than generic prompting,
using a longitudinal calcium dataset as a case study.

## Data Sources
- **Airport traffic data**: Scraped from Wikipedia's
  [List of busiest airports by passenger traffic](https://en.wikipedia.org/wiki/List_of_busiest_airports_by_passenger_traffic)
  (publicly accessible).
- **Calcium dataset**: calcium.csv - a longitudinal dataset of ulnar calcium
  measurements across two treatment groups (Null vs. Diet/Exercise), provided
  as course material.
